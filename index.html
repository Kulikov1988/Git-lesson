  • git clone path to the repository
  • git status - статус текущего локального репозитория
  • git add path-to-file - добавить новый файл под контроль ГИТА
  • git commit -am "add new file index.html" - зафиксировать изминения и дать им коментарий
  • git push - отправить все закомичинный изминения в репозиторий на сервер
  • git pull - забрать свежие обновления
